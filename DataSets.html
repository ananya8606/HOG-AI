DataSet Generation¶

It is a process of creating training and validation data for a neural network.We define the inputs and the corresponding correct output of all possible cases that we can generate.¶

DataSet¶

Numpy arraya in general consisting of input valus in one and corresponding output values shaped according to the network architecture in another.¶

import numpy as np
from keras.models import Sequential
from keras.layers import Activation,Dense
import matplotlib
import matplotlib.pyplot as plt

location=input("Enter the path of location to get the data:")
print(location)

tmp=np.load(location+"\datafile.npz")

data=tmp['x']
out=tmp['y']

model = Sequential()
model.add(Dense(16, input_shape=(2,)))
model.add(Activation('relu'))
model.add(Dense(32))
model.add(Activation('relu'))
model.add(Dense(16))
model.add(Activation('relu'))
model.add(Dense(1))
model.add(Activation('sigmoid'))

model.compile(loss='mean_squared_error',
              optimizer='adam',
              metrics=['accuracy'])
model.summary()

history=model.fit(data,out,epochs=500)

plt.plot(history.history['accuracy'])
plt.title("Model Accuracy")
plt.show()

plt.plot(history.history['loss'])
plt.title("Model loss")
plt.show()
